# 🚀 UGC AI – AI Content Generation Platform

UGC AI is a full-stack AI powered platform that allows users to generate user-generated content such as images, videos, and creative assets using artificial intelligence.

The platform provides a clean UI for generating content, managing prompts, and exploring creations from the community.

---

## 🌐 Live Demo

🔗 **Live Website:**
https://ugc-project-harsh.vercel.app/

---

## ✨ Features

* 🤖 AI powered content generation
* 🖼️ Image and media generation using prompts
* 👤 Secure user authentication
* 🌍 Community section to explore generated content
* ⚡ Fast and responsive UI
* ☁️ Cloud deployment

## 🛠️ Tech Stack

### Frontend

* Node.js

## ⚙️ Environment Variables

Create a `.env` file inside the **client** folder:

```
VITE_BASEURL=your_backend_url
VITE_CLERK_PUBLISHABLE_KEY=your_clerk_publishable_key
```

## 🚀 Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/harshgavandit/ugc-project.git
```

### 2️⃣ Navigate to the project

```
cd ugc-project
```

### 3️⃣ Install dependencies

Frontend:

```
cd client
npm install
```

Backend:

```
cd ../server
npm install
```

---

### 4️⃣ Run the project

Frontend:

```
npm run dev
```

Backend:

```
npm run start
```
